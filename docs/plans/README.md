# Plans

This directory is for **active** PRDs and execution plans only. There
are no active plans right now: every v0.2 / v0.3 / production-polish
PRD has shipped or been superseded and lives in
[`../archive/plans/`](../archive/plans/).

When you start a new feature that needs a written plan, drop the
`.prq.md` file here. Move it to `../archive/plans/` (and stamp
`status: archived` + `updated: <today>` in the frontmatter) when the
feature ships or is killed.

## Where to look instead

- Current branch state and recent releases: [`../STATE.md`](../STATE.md)
- Post-1.0 polish runway and partial systems: [`../PRODUCTION.md`](../PRODUCTION.md)
- Long-horizon ambition (multiplayer, expansions): [`../DESIGN.md`](../DESIGN.md) "Future Direction"
- Pre-store-submit checklist: [`../LAUNCH_READINESS.md`](../LAUNCH_READINESS.md)
- Tag-and-publish runbook: [`../RELEASE.md`](../RELEASE.md)
