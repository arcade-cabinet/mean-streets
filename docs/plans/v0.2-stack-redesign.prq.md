# Feature: v0.2 Stack Redesign

## Overview

Full game rewrite. Replaces the v0.1 quarter-card/backpack/runner model
with a stack-based turf war. All v0.1 combat, deckbuilding, and UI is
obsolete. Creative writing (names, archetypes, affiliations, taglines)
carries forward. ONE giant PR, web-only deployment.

**Authoritative specs**:
- `docs/RULES.md` (updated) — mechanics
- `docs/DESIGN.md` (updated) — vision

**Branch**: `feat/v0.2-stack-redesign`

## Constraints

- pnpm only (no npm/yarn)
- Biome lint (no ESLint)
- Max 300 LOC per file
- Capacitor 8 for Android/iOS, jeep-sqlite, Vitest, Playwright, Vite 8
- Tests: node + DOM + browser + e2e — all must pass before merge
- No debug APK firing on merge; web deploy via cd.yml on push to `main`
- Squash-merge

### Epic C — Sim Engine Rewrite

- [ ] P1: Rewrite `src/sim/turf/board.ts`:
  - `createTurf()`, `addToStack(turf, card)`, `removeFromStack(turf, idx)`
  - `positionPower(turf)`, `positionResistance(turf)` — aggregate over stack, respect `sickTopIdx`
  - `turfAffiliationConflict(turf, incoming)` — check rival graph with buffer rule
  - `seizeTurf(defender, defenderTurfIdx, attacker)` — remove turf, transfer modifiers to attacker's chosen turf
- [ ] P1: Rewrite `src/sim/turf/attacks.ts`:
  - `resolveDirectStrike(attackerTurf, defenderTurf) → StrikeResult` (kill | sick | busted)
  - `resolvePushedStrike` — spend 1 currency from attacker turf, +denomination/100 power, splash sicks one beneath
  - `resolveFundedRecruit` — sum currency on attacker turf vs target resistance × affiliation multiplier
  - Apply `strike-bottom` / `strike-anywhere` abilities (Shark, Ghost, legendaries)
  - Transfer modifiers on kill with affiliation re-eval on transfer
- [ ] P1: Rewrite `src/sim/turf/environment.ts`:
  - `stepAction(state, action)` with action types: `play_card`, `direct_strike`, `pushed_strike`, `funded_recruit`, `discard`, `end_turn`
  - Draw-gate enforcement: reject `play_card(modifier)` when `toughsInPlay === 0`
  - First-turn action budget (5) vs normal budget (3/4/3 by difficulty)
- [ ] P1: Rewrite game loop in `src/sim/turf/game.ts` (or equivalent):
  - `createMatch(config: GameConfig) → MatchState`
  - `runTurn(state, actions) → MatchState`
  - `isGameOver(state) → winner | null` — winner = opponent with 1+ turfs when other has 0
